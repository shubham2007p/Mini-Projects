# Simple Student Management System CLI

A lightweight, interactive Command Line Interface (CLI) application written in Python to easily manage student records. This mini-project allows users to dynamically add, remove, and search for students in an in-memory database using their Roll Numbers.

## Features

* **Add Student:** Interactively input and verify student details (Name, Roll no., Branch, CGPA) before saving them to the database. Includes a review step to edit details on the fly.
* **Remove Student:** Delete a student's record quickly by searching for their unique Roll Number.
* **Search Student:** Instantly pull up a student's information using their Roll Number.
* **Input Handling:** Basic error handling to ensure invalid menu choices do not crash the program.

## Prerequisites

* Python 3.x installed on your machine. No external libraries are required.
