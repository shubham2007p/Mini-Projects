# 💰 Simple Expense Tracker

A lightweight, command-line Python application to manage and track your expenses. 

## ✨ Features

This script provides an interactive menu with the following capabilities:
* **Add Expenses:** Quickly input new expense amounts.
* **View Expenses:** See a running list of your current session's expenses.
* **Total Expense:** Calculate the total sum of all entered expenses instantly.
* **Delete Expense:** Remove a specific expense amount from your list.
* **Save Records:** Export your current expense list to a local text file (`expense.txt`).
* **Read Records:** Read and display saved expenses from the text file.

## 🚀 How to Run

1. Ensure you have Python installed on your system.
2. Open your terminal or command prompt.
3. Navigate to the folder containing the script.
4. Run the script using the following command (replace `main.py` with your actual file name if it's different):
   ```bash
   python main.py
