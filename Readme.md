# 🧪 Mini-Projects & Learning Sandbox

Welcome to my learning sandbox! This repository is a collection of bite-sized experiments, standalone scripts, and mini-projects. 

> **💡 The Philosophy:** This is a *learning* journey, not a *building* journey. The focus here is on understanding how things work under the hood, testing new libraries, and grasping specific concepts rather than creating perfect, end-to-end applications. Expect to see messy code, print statements, and lots of comments!

## 🎯 Objectives
* To create a low-pressure environment for trying out new technologies.
* To break down complex concepts into isolated, testable scripts.
* To document my "aha!" moments and key takeaways for future reference.
